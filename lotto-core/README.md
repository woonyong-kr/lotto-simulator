# lotto-core

모든 모듈에서 공유하는 도메인 로직과 유틸리티입니다.

## 역할

로또 시스템의 핵심 규칙을 정의하고, 공통으로 사용하는 값 객체와 응답 형식을 제공합니다.

---

## 구현 항목

### 로또 번호 모델
- [x] 1-45 범위 검증하는 값 객체
- [x] 6개 번호를 묶는 일급 컬렉션
- [x] 중복 방지 및 정렬 보장
- [x] 불변 객체로 설계

### 당첨 판정
- [x] 일치 개수로 등수 계산
- [x] 보너스 번호 포함 여부 판단
- [x] 당첨 금액 정보

### 번호 생성
- [x] 무작위 6개 번호 생성
- [x] 중복 없이 자동 정렬
- [x] 스레드 안전성 고려

### 공통 응답 구조
- [x] 성공/실패 구분 가능한 형식
- [x] 제네릭 타입 지원
- [x] 타임스탬프 포함

### 예외 체계
- [x] 비즈니스 예외 기본 클래스
- [x] 도메인별 구체적 예외
- [x] 에러 코드 관리

### 상수
- [x] 로또 규칙 관련 상수
- [x] 시스템 전역 상수
