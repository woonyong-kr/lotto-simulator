# lotto-bot-client

봇 인스턴스를 메모리로 관리하는 클라이언트입니다.

**포트:** 8200  
**DB:** 없음

## 역할

중앙 서버에서 봇 정보를 받아 메모리에 저장하고,
주기적으로 구매를 실행하며 연결 상태를 관리합니다.

---

## 구현 항목

### 메모리 관리
- [ ] 봇 인스턴스 저장소
- [ ] POS 연결 정보 관리
- [ ] 동시성 제어

### 연결 상태 관리
- [ ] 헬스 상태 추적
- [ ] 실패 카운트 관리
- [ ] 상태 전환 로직

### 자동 구매
- [ ] 주기적 스케줄링
- [ ] 비동기 처리
- [ ] 실패 시 재시도

### 헬스체크
- [ ] POS 터미널로 주기적 핑
- [ ] 응답 시간 수집
- [ ] 실패 처리

### 재할당
- [ ] 문제 감지
- [ ] 연결 제거
- [ ] 새 터미널 요청
- [ ] 실패 시 봇 제거

### API 제공
- [ ] 봇 생성/조회/삭제 엔드포인트
- [ ] 오너 검증 엔드포인트
- [ ] 상태 조회 엔드포인트
