# lotto-dashboard

시스템 전체를 모니터링하는 대시보드입니다.

**포트:** 8500

## 역할

각 모듈에서 데이터를 수집해서 통합하고,
실시간으로 클라이언트에 전달합니다.

---

## 구현 항목

### 데이터 수집
- [ ] 중앙 서버 정보 수집
- [ ] 봇 클라이언트 정보 수집
- [ ] POS 매니저 정보 수집
- [ ] 병렬 호출 처리
- [ ] 부분 실패 허용

### 데이터 통합
- [ ] 여러 소스 병합
- [ ] 일관성 검증
- [ ] 누락 처리

### 실시간 전송
- [ ] SSE 스트림 관리
- [ ] 클라이언트 구독 관리
- [ ] 이벤트 푸시
- [ ] 타임아웃 처리

### 캐싱
- [ ] 데이터 캐시
- [ ] TTL 관리
- [ ] 자동 갱신

### API 제공
- [ ] 전체 요약 엔드포인트
- [ ] 회차 정보 엔드포인트
- [ ] 봇 상태 엔드포인트
- [ ] POS 상태 엔드포인트
- [ ] SSE 스트림 엔드포인트
