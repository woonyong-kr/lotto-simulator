# lotto-pos-manager

터미널 풀을 관리하는 매니저입니다.

**포트:** 8300

## 역할

터미널을 등록받아 풀로 관리하고, 요청이 오면 할당해줍니다.
할당된 터미널의 상태를 주기적으로 확인합니다.

---

## 구현 항목

### 터미널 풀 관리
- [ ] 사용 가능 풀과 할당 풀 분리
- [ ] 등록/할당/해제 로직
- [ ] 중복 등록 방지
- [ ] 동시성 제어

### 상태 확인
- [ ] 주기적인 헬스체크
- [ ] 응답 시간 측정
- [ ] 실패 횟수 추적
- [ ] 임계값 초과 시 제거

### 할당 전략
- [ ] 사용 가능한 터미널 선택
- [ ] 풀 비었을 때 예외 처리
- [ ] 해제 시 풀 복귀

### API 제공
- [ ] 터미널 등록 엔드포인트
- [ ] 터미널 할당 엔드포인트
- [ ] 터미널 해제 엔드포인트
- [ ] 풀 상태 조회 엔드포인트

---

- [ ] 할당 후 사용 불가 상태 확인

- [ ] 동시 할당 요청 처리
