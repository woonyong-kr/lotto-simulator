# lotto-pos-terminal

티켓 구매를 실제로 처리하는 워커입니다.

**포트:** 8400+

## 역할

오너를 관리하고 구매 요청을 중앙 서버로 전달합니다.
헬스체크에 응답하고 응답 시간을 수집합니다.
